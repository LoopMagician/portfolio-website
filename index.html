  • Asteroid Defense

    • This is another project that I created in Unity 6. It is the first game that I have created on my own.

    Here is a full description of the games development process:

    This was a two week sprint. During the sprint, I created tasks for myself to implement into the game. In particular, I had to create the environment which included the city game object. I also had to create the asteroid prefeb and its movement script which also controlled destroying the object on user click. This script was linked to another script that would spawn the asteroids at random intervals and increase the difficulty by increasing the speed and spawn rate. After This was complete, I was faced with a design decision for the flow of gameplay. The city had to have a health status, but an object of that type wouldn't suit well with something as simple as a health bar. Instead, I added a crumbling effect that lowered the city off the screen during each hit. After the last hit, the city will continue to crumble until it is completely off the screen, in which the game will end. I also created a score, start menu, and game over menu ui to accompany the game. Once this was done, the core of my game had been developed. At this point it was time to add the 'juice' to the game. This included things such as explosion effects, sound effects, background music, and a crosshair for the player. This is the jist of what the game currently is. I hope you enjoy it.
